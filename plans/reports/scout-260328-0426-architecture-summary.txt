PROCESSING PIPELINE
  Input: VideoCapturer.read() [cv2.VideoCapture]
  Detect: face_analyser.get_one_face() [buffalo_l ONNX]
  Swap: face_swapper.process_frame() [inswapper_128_fp16.onnx]
  Enhance: face_enhancer.process_frame() [gfpgan-1024.onnx] (optional)
  Mask: face_masking.process_frame() (optional)
  GPU: gpu_processing.* [CUDA or CPU fallback]
  Output: 
    Live: virtual_camera.send() -> v4l2loopback/OBS
    Batch: ThreadPoolExecutor -> ffmpeg encode -> MP4/MKV

Threading:
  - Main thread: GUI event loop
  - Live capture: Implicit cv2.VideoCapture thread
  - Batch workers: ThreadPoolExecutor (batch_size = min(32, N_frames/workers))
  - Synchronization: MAP_LOCK for source_target_map
